The Demo Portfolio is a school project designed to showcase creative web design skills using HTML and CSS. This portfolio features smooth animations, engaging transitions, and visually appealing layouts , demonstrating the power of CSS animations in modern web design.

As part of the project, the portfolio highlights different sections with dynamic motion effects, making the browsing experience more interactive and visually striking. The goal was to experiment with hover effects, keyframe animations, and responsive design , ensuring a polished and professional presentation.

This project not only demonstrates technical proficiency in front-end development but also serves as a creative expression of design and animation skills.
