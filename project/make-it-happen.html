MakeItHappen is a task management web application designed to help users stay organized and productive. Built using HTML, CSS, and JavaScript, the app features a clean and responsive interface, making it easy to add, update, and delete tasks.

The app includes secure user authentication with login and registration features, ensuring that each user's tasks are kept private. Backend functionality is powered by Knex.js lowing for efficient interaction with the database to store and retrieve user data and tasks.

Knex.js is a "batteries included" SQL query builder for PostgreSQL, CockroachDB, MSSQL, MySQL, MariaDB, SQLite3, Better-SQLite3, Oracle, and Amazon Redshift designed to be flexible, portable, and fun to use.

With the implementation of Express.js, MakeItHappen ensures fast, secure, and scalable handling of requests. Whether you're managing a to-do list for work, school, or personal projects, MakeItHappen is the ideal tool to keep you on track and organized.
