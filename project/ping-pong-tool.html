Project Overview

The Ping Pong Tool was developed to modernize a local ping pong club's operations, replacing their inefficient paper-based drink ordering system with a comprehensive digital solution. This full-stack web application manages everything from member authentication and match scheduling to financial tracking and payment processing.

Technology Stack & Architecture

Built with Node.js and Express.js for the backend, the application uses Objection.js ORM with Knex.js for database operations and SQLite for reliable data storage. The frontend combines EJS templating with vanilla JavaScript and responsive CSS, ensuring optimal performance across all devices.

Security is handled through JWT authentication with bcrypt password hashing, while the modular architecture allows for easy maintenance and scalability as the club grows.

Core Features & Functionality

The system includes a complete user management system with role-based access control, allowing administrators to manage members, guests, and permissions. Members can track their match attendance, view team schedules, and participate in player selection for upcoming games.

The digital ordering system features a product catalog with real-time stock management, shopping cart functionality, and integrated payment processing through Mollie. This eliminates the club's previous paper-based ordering problems and provides accurate financial tracking.

Administrative Dashboard

Administrators benefit from a comprehensive management panel that handles user accounts, financial reporting, and system configuration. The dashboard includes data export functionality for record keeping and a notification system for member communication.

Match management features allow importing schedules from ICS files, tracking attendance in real-time, and managing team assignments efficiently, replacing manual scheduling processes.

Payments

The application integrates Mollie payment gateway for online transactions while also supporting cash payments for in-person purchases. Order statuses update automatically based on payment confirmation, providing seamless financial management.
