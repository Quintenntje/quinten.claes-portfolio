Visit the website

Cooking Master is a recipe website that offers a clean, user-friendly design built with HTML, CSS, and JavaScript. It features a robust CRUD functionality using a JSON file to manage recipes, allowing users to create, read, update, and delete recipes easily.

Powered by Express.js, the website uses REST API calls to fetch and update recipe data, ensuring fast and secure data handling. Whether you're exploring new recipes or managing your own, Cooking Master provides a seamless, modern experience for all cooking enthusiasts.
